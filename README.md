# th-py
A colorful terminal editor developed with pure Python

It only relies on text-discoloration 2.7.2, without any other additional third-party libraries

## Version
v1.0.0

## Features
- Random colored >>> command prompt
- Typewriter startup animation and elegant dividing lines
- Support writing Python code line by line
- Enter `end` to stop input and run code automatically
- Automatically clean up temporary files after abnormal exit

## Installation
```bash
pip install th-py
```
